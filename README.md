# Hangman Game

A web-based Hangman game built with Flask and vanilla JavaScript.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
python app.py
```

3. Open browser and navigate to:
```
http://localhost:5000
```

## How to Play

- Click on letters to guess them
- You have 6 lives
- Use the "Get Hint" button to reveal a random letter
- Click "New Game" to start a new game
